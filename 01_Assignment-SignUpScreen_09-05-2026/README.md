# 🏥 NovaMed - Sign-Up Screen UI

A premium, dark-themed Sign-Up interface built with **React Native** and **Expo**. This project showcases advanced UI techniques including linear gradients, SVG integration, and responsive layouts for the **NovaMed** health platform.

## ✨ Features
- **Modern Dark UI:** Uses `expo-linear-gradient` for a sleek background effect.
- **Custom Branding:** Integrated SVG logo and custom typography using NovaMed's signature neon-green theme (`#C8FF00`).
- **Form Handling:** Optimized input fields with `KeyboardAvoidingView` and `ScrollView` for seamless mobile interaction.
- **Social Integration:** Custom-styled buttons for Google, Facebook, Instagram, and Apple authentication.
- **Cross-Platform:** Designed to scale beautifully across Web, iOS, and Android.

## 🛠️ Technical Stack
- **Framework:** [Expo](https://expo.dev) (SDK 55)
- **Icons:** `@expo/vector-icons` (AntDesign, FontAwesome5, MaterialCommunityIcons)
- **Gradient:** `expo-linear-gradient`
- **SVG:** `react-native-svg`
- **Layout:** Flexbox with `react-native-safe-area-context`

## 🚀 Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start the Project**
   ```bash
   npx expo start
   ```

3. **Development File**
   The main logic and styling for this screen can be found at:  
   `src/app/index.tsx`

## 📁 Key Components Used
- `LinearGradient`: For the primary background aesthetic.
- `Pressable`: For high-performance touch feedback on buttons.
- `TextInput`: Styled with custom containers and inline icons.
- `KeyboardAvoidingView`: To ensure the UI stays accessible when the keyboard is open.
